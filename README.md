# Retail Basket Analysis using Association Rule Mining

## Project Overview
This project analyzes customer purchase transactions using the Apriori Algorithm to discover products frequently bought together.

## Dataset
Online Retail Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Methodology
1. Data Cleaning
2. Transaction Basket Creation
3. Frequent Itemset Mining
4. Association Rule Generation
5. Business Insights

## Results
- 11,624 Transactions
- 522 Frequent Itemsets
- 773 Association Rules

- Support
- Confidence
- Lift
